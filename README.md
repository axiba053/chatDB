# chatDB
**利用文件建立永久数据库，结合chatgpt提问，实现专业客服功能。**

界面如下：
![girl](./image/jiemian.png)

## 安装说明

1. 安装依赖
```commandline
pip install -r requirements.txt
```

2. 填入API
```
打开.env文件，在下面填入自己的API
OPENAI_API_KEY='填入你自己的API'
```   

3. 运行
```commandline
streamlit run chatDB.py
```
